Day 1

1. [多选] Golang通过plugin.(*Plugin).Lookup函数可以查找到插件定义的哪些东西？

  1. 变量
  2. 函数
  3. 类型
  4. 包
答案&解析

a、b；

a和b都是能被赋值给interface{}类型的变量，但是c和d不能。因此Lookup方法返回的结果是一个interface{}类型（Symbol类型），因此c和d不能通过Lookup返回。

参考Lookup方法的说明： func (p *Plugin) Lookup(symName string) (Symbol, error) Lookup searches for a symbol named symName in plugin p. A symbol is any exported variable or function. It reports an error if the symbol is not found. It is safe for concurrent use by multiple goroutines.
